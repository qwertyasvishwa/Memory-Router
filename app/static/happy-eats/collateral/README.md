## üì§ Converting to PDF

**Method 1 - Browser:**
1. Open HTML file in Chrome/Edge
2. Press Ctrl+P (Print)
3. Select "Save as PDF" as destination
4. Click Save

**Method 2 - Online Tools:**
- Use htmltopdf.com or similar services
- Upload HTML file
- Download generated PDF
